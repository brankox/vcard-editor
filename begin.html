vCard Object

A vCard data stream may include one or more vCard objects. An individual vCard object is identified within a data stream by the appearance of the Begin vCard Delimiter:

BEGIN:VCARD

The sentinel string must appear as the first characters in the data stream or the first characters on a line.

The vCard object is terminated with either the logical end of the data stream or the appearance of the End vCard Delimiter as the first character on a line:

END:VCARD
